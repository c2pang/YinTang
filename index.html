前言

从来没有想过我的祖先来自哪里。我的橱柜里有一份 1992 年家谱记录的几页复印件。它在那里放了很多年，直到2017年，我想知道我的曾祖父母是谁？开始了我的寻根历程。

当我拿到1992年完整的族谱时，寻根之旅才真正开始。我原本希望从族谱中整理出祖先的故事，但未能如愿。明朝之前的历史无法无缝衔接，或许是因为我的中文能力有限。在提升语言能力的同时，我也回到海南故乡进行实地考察。对当地地名的熟悉以及墓碑上的文字给了我不少线索，但仍未能完全解开我心中的疑惑。于是，我决定借助基因技术来填补明朝之前甚至更早的历史空白。

2020年9月完成第一次基因检测后，我对Y染色体承载的信息产生了浓厚兴趣。父系溯源依赖Y染色体突变的代际传递，通过对比不同检测者之间的突变差异，可以推测共同祖先存在的年代（如图一）。经过几年的学习和研究，我终于读懂了这本『无字行书』。当找到基因上的宗亲时，那种感觉宛如与失散多年的至亲重逢！

三种祖先

我寻祖的方针是从三个层面着手，即基因祖先、姓氏祖先、文化祖先。

  • 文化祖先如三皇五帝，概括五千年。
  • 姓氏祖先如冯姓，可靠性最早到宋朝。
  • 基因祖先，可追溯至20万年前，且可靠性高。

人类的演变

  • 约700万年前人科动物与黑猩猩分离，开始直立行走。
  • 约400万至200万年前南方古猿如著名的“露西”。
  • 约20万年前至今，解剖学上与现代人相同的现代智人开始发展出语言、艺术和复杂社会结构。
  • 约7万至5万年前，现代智人走出非洲，从非洲扩散至全球，取代其他古人类。
  • 约1万年前的地球正处于气候变暖、冰川退缩和海平面上升的时期。这一时期的环境变化为人类从狩猎采集向农业社会的过渡提供了条件，标志着人类文明发展的重要转折点。农业革命改变了人类，从狩猎采集转向农业，形成定居社会。
  • 约5000年前至今，文明发展，文字、城市、国家和复杂社会结构的出现，推动科技和文化进步。

明朝•隐塘的缘起

隐塘始祖

明•景泰1454年，八世祖冯元佑在万泉河口博鳌当汛守，遂家於此，生四男（如图二）。长子文俊肇基汀洲，次子文昌肇基培兰村，三子文盛迁居沙美村，四子文成留在隐塘。

隐塘村

隐塘村（图三）坐落在万泉河口，美如增江作的一首七言（图四）。

长支祖

基因/血缘祖先

文俊公与其弟在隐塘鼎新祖庙一座，坐酉向卯。主修上代谱系后迁居汀州。文俊公有四位曾孙：孔宽、孔盛、孔兴、孔臭。

  • 孔兴支系
    • 曾孙辈：有会的儿子天禄(亦名天桂)（移居横岭村，有待考察）
    • 曾孙辈：有宣（移居沙美村）
  • 孔臭支系
    生三子：宗泰、宗清、宗源。
    • 宗泰分支
      • 孙辈：元遇、元会（兄弟同心，迁于乔木）
        • 长孙元遇 → 迁乐美村
        • 次孙元会 → 迁陈村
    • 宗清分支
      • 长子一豪 → 移居沙头坡
      • 三子一明 → 移居石富埇村
    • 宗源分支
      廪生宗源于明万历年间传上代谱系。
      (注：廪生是明代科举制度中享受津贴的生员)
      • 长子一滋 → 迁江水村与仰止村
      • 次子一凤 → 留居汀州

次支祖

基因/血缘祖先

文昌公的后裔主要定居在培兰村。文昌公的儿子冯补曾任八品修职郎，其孙冯修被授予千户之职。冯修有两位曾孙：汝元和汝隆。

  • 汝元育有三子：
    • 长子仁挺，敕授文林郎，后移居龙潭村；
    • 次子仁富的后裔迁居南港村；
    • 三子仁升迁居沙美村。
  • 汝隆育有四子：
    • 长子高龙，恩赐登仕郎，留守培兰村；
    • 次子高盛的后裔移居朝锡村；
    • 三子高明的后裔分迁南港村与和顺村；
    • 四子高才的后裔移居加昌村。

三支祖

基因/血缘祖先

文盛公的后裔主要定居在沙美村，少数移居至北山、烟墩、万州。文盛公的玄孙冯先育有二子裔德和明德。

四支祖

基因/血缘祖先

文成公的后裔主要留在隐塘村，大多分居在清调村与附近的村庄，少数移居至万州、乌坡、龙滚、阳江等地。

文成公有三位曾孙：德全、德明、德贤。德全生世豪、世亮两子。世豪的长子廷侯迁居清调村，次子廷宰留居隐塘。德明的孙子廷仁移居博鳌，德贤的孙子廷佐则迁至竹史箩村。

世豪公的墓是现存最古老的祖墓，曾从大灵岭迁至后埇尾岭，最终迁至现在的公墓。

廷宰公嫡长孙联章的后裔迁居万州，祖辈多安葬于万宁宝塘。据悉，宝塘村西南约200米处便有冯家村，我计划找个时间前去实地探访，以期寻回失联多年的宗亲。

基因验证

为了消除对基因检测的疑虑，2023年，我和增贵在中国做了基因检测。结果显示，我们有一位共同祖先生活在1434年左右，这与族谱中有明确出生年份记载的元佑公非常接近，误差仅19年。

科学的进步为我们寻根问祖打开了新的窗口。2025年6月，吉林大学隋明儒的硕士论文《基于古DNA技术的Y染色体分析在法医遗传学中的应用研究》[1]，通过古DNA证据确认明太祖朱元璋的基因类型[2]为O-MF12415。这一发现再次印证了基因检测技术的精准性与可靠性。这让我们对基因溯源技术充满了信心。

我们将启动一项意义非凡的计划：运用基因检测技术，为族谱中“失叙名录”（失考支系）的宗亲们，找回失落的血脉记忆。我们深知此事关系重大，因此慎之又慎。在每一位家人参与检测之前，我们必须周全地安排好所有可能情况的应对方案，尤其是未匹配时的沟通与安抚。我们殷切期盼着每一次阳性的匹配结果，那将是我们携手深入考证、厘清共祖脉络的开始，共同为家族补全这块珍贵的血脉拼图。

元朝•冯周旺

四世祖冯周旺在元朝时，由琼来乐邑。因文场不遂，武闱标名，官任乐汛，遂肇基於乐土。

宋朝•冯细歌

二世祖冯细歌在宋朝时，因官任琼府而从福建莆田迁居海南琼山之墩头村。

隋朝•福建祖

基因/血缘祖先

福建祖在隋•开皇(584年)时从莆田分四支:

  • 1支，明朝时北迁到江苏、山东(张氏)
  • 2支，留在福建(徐、石、袁氏)
  • 3支，唐•贞观十八年(644年)时分两支，开始南迁到潮州(林氏)、海南(冯氏)
  • 4支，明朝时北迁到辽宁(王氏)
  • 有两位因为没深度检测而无法归入支系，分别是台湾(汪氏)、广东(陈氏)

根据基因资料，春秋时期(前635年)以来，我族基因检测的人数不多，到2025年为止只有31人，而且都分布在中国的沿海城市，从海南到黑龙江。

到目前为止的基因资料显示春秋周襄王时共分出两支。其中一支即以上的福建祖，另一支有江苏(吕氏)及福建(陈氏)。以下是23魔方祖源数据库绘制的隐塘春秋祖先的后裔分布情况:

春秋战国•河南荥阳冯城

姓氏祖先

冯氏最初发祥于河南省荥阳(XíngYáng)，以下有几种说法：

  • 毕万奉禄食采地于冯城
    • 据《元和姓纂》、《后汉书》所载。
    • 姬姓，毕氏，周文王庶子毕公高的后裔，春秋时期晋国的大夫(晋献公，？~前651年)
    • 冯城一说是陕西省大荔县的冯翊城，一说是河南省荥阳县西
  • 出自归姓(东夷中有一支归夷)，前544年春秋中后期周景王时，郑国名卿冯简子(归姓冯氏第一人)封地冯邑
    • 据《世本•姓氏篇》记载，入居冯邑后，遂号冯简子
    • 他的封地冯邑，正是后来并入魏国的冯城
  • 约前445年春秋战国时期
    • 魏国国君魏文侯名斯，也是毕万的后代
    • 魏斯之子魏启字长卿，封于冯 (今河南省荥阳县西)
  • 前403年周威烈王再封毕万的支孙冯文孙于冯城

周朝•始平郡

文化祖先

冯氏的先祖从姬姓开始。周文王(西伯侯，姬昌，约前1152年 ~ 前1056年)第十五子(庶子)毕公高封地在毕，故子孫以毕为姓。 西周时，毕公高后裔毕万在晋为大夫，当时晋献公将其地分封给有功之臣，毕万被封于魏(今山西芮城县附近)。

引用维基百科

晋献公去世后(前651年)，四子争立为国君，引发内乱。毕万生芒季，芒季子魏武子(名犨)跟随公子重耳流亡十九年，是五贤之一，返国后继承魏氏。

冯姓产生的过程看来好像是姬 → 毕 → 魏 → 冯。

始平考证

引用维基百科

始平郡，中国西晋(266~316年)时设置的郡。治所在槐里县 (今陕西省兴平市东南5公里南佐村)。

大汶口文化

存在于前3500年至前2500年。主要分布在山东省泰山附近地区，南至江苏北部地区。

大汶口文化[5]与南岛民族存在关连。隐塘宗亲与南岛人(O-AM01847)的共同祖先(O-AM01856)在一万五千年前。推测南岛人可能在大约8千年前从中国东南沿海外迁。

大汶口文化与东夷/夷人[6]有着密切的关系。研究东夷有助于了解隐塘上古的祖先。我自知没能力做这方面的研究，只好多关注东夷的研究报告了。

后记

在查阅1992年编撰的《原谱失叙综合本》时，发现因年代久远、支系散居，有十四支宗亲失去联络而被列入失叙名录。如今基因技术已能精准识别血缘亲疏并推算共祖年代，我希望能借助科技之力，帮助这些失散多年的宗亲重归祖脉。愿这份寻根溯祖的精神，能够薪火相传，生生不息。
